# const

- 修饰普通变量的值

  ```C++
  const int a = 7;
  ```

- 修饰指针变量

  - 修饰指针指向的内容

    ```c++
    const int *p = 8;
    ```

    指针指向的内容不变，指针指向的地址可以改变（不可以通过指针改变指向的内容）

  - 修饰指针

    ```c++
    int* const p = &a;
    ```

    指针指向的地址不变，指针指向的内容可以改变

  - 同时修饰两者

    ```C++
    const int * const  p = &a;
    ```

- 修饰函数

  - 使用const修饰传递的参数

  - 使用const修饰传递的指针

    防止指针被篡改

  - 使用const修饰函数
  
    ```c++
    int get_cm() const
    {
       return _cm;
    }
    ```
  
    保持对象中的成员内容不变
