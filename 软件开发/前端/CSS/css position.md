# [css position](https://www.w3school.com.cn/css/css_positioning.asp)

- static

  默认定位方式，根据页面的正常流定位

- relative

  相对于其正常位置（static）定位

- fixed

  在窗口中固定位置，即使滚动页面也不会发生改变

- absolute

  在父组件中固定位置

- sticky

  根据用户的滚动位置进行定位

## 重叠元素

使用z-index属性指定元素的堆叠顺序
